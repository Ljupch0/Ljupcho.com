AEX Amsterdam Stock Index

Dashboard which presents daily data of the AEX Amsterdam stock index. It's built with shinydashboard and pulls data by scraping from finance.yahoo.com, wikipedia.org, and other websites. The dashboard features several interactive visualizations such as d3treeR, Datatable, and a dygraph presenting historical data.

Net Promoter Score Dashboard

This dashboard presents data obtained from an NPS survey. The data is taken from this google sheet and calculates the NPS score, it's distribution over time, and other demographic data. You can add data to the spreadsheet to see the dashboard update.
